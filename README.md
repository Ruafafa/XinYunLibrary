## 📋简介

芯云图书馆是云顶书院七期芯云项目组的寒假团队协作实战项目，旨在体验团队前后端开发协作流程，并对寒假研学成果进行验收。

该图书借阅系统中的各个模块来自每个XinYuner的精雕细琢，采用敏捷开发模式开发；
